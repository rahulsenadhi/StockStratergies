## Why a new table component is the keystone

`st.dataframe` (17 uses) renders in a canvas locked to Streamlit's config `theme.base=dark` â€” CSS cannot retheme it, so it shows black boxes in light mode (this forced dark-only earlier). Plotly-tables and raw HTML tables also look dated. Replacing all three with ONE modern, CSS-themed HTML table component fixes "outdated tables" AND unblocks light mode in a single move. HTML tables are also lighter than Plotly-tables â†’ small perf win.

## Components

### 1. Theme system (`core` or in-module)
- Re-add sidebar Light/Dark toggle (sets `data-theme` via `st.html` JS â€” pattern already exists, was removed).
- Semantic CSS tokens for both themes: `--surface`, `--surface-2`, `--border`, `--text`, `--text-muted`, `--accent`, status `--pos/--neg/--warn`. Light + dark token sets under `[data-theme="light"]` / `[data-theme="dark"]`.
- Convert remaining inline dark-hex HTML cards to use these tokens/classes so they flip.
- Plotly: `_plotly_base()` already theme-aware (done). Extend to gridline/axis colors per theme.
- **Perf:** pure CSS, zero runtime cost.

### 2. `_modern_table(df, *, numeric_cols=None, status_col=None, sortable=False, max_height=None)`
- Renders semantic HTML `<table>` with CSS classes (no inline hex).
- Styling: subtle bottom-border rows (no heavy grid), hover highlight, sticky header, compact padding (6â€“8px), `font-variant-numeric: tabular-nums` on numeric cells (right-aligned), IBM Plex Mono for numbers.
- `status_col`: render pill badges with semantic colors.
- `sortable=True`: emit a small vanilla-JS sort (click header â†’ sort rows); used only on big tables.
- `max_height`: CSS `overflow-y:auto` + sticky header for scroll on long tables.
- Theme-aware via tokens â†’ works in light and dark.

### 3. Migration
- Replace all 17 `st.dataframe` + `chart_plotly_table` + raw HTML `<table>` call sites with `_modern_table`.
- Big tables (50-stock rankings, trades logs): `sortable=True, max_height=...`.
- Small tables (Recent Backtests, signals, KPI breakdowns): static.
